inlinestatic

Visit types (not values) of the tuple, passing each element as single template argument. Unlike for_each_single, produces tuple from callback's return value, respecting argument order.

This helper is especially useful with aui::member<>::args trait:

void MyClazz::myMethod(int arg1, std::string arg2);
aui::tuple_visitor<aui::member<&MyClazz::myMethod>::args>::for_each_make_tuple([]<typename T>() {
return readValue<T>();
}); // -> std::tuple<int, std::string>
